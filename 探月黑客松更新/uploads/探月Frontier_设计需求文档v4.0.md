## 3. 行星意象系统（**v4.0 新增核心**）

### 3.1 设计哲学

探月（Moonbot）的品牌核心是月球。本次赛事的视觉语言扩展到**整个太阳系**——让"Frontier 边界"的宇宙叙事真正成体系。

每颗行星对应赛事中的一个核心元素，构成视觉锚点矩阵。

### 3.2 行星语义映射

| 行星 | 中文 | 视觉用途 | 寓意 |
|---|---|---|---|
| 🌑 月球 | 月球 | **Hero 主视觉** / 探月品牌锚点 | 探月之根，整场赛事的起点 |
| ☉ 太阳 | — | 不直接出现，作为隐含中心 | — |
| ☿ 水星 | 水星 | **AI Agent 与软件** 赛道标识 | 速度、敏捷、信息流 |
| ♀ 金星 | 金星 | **陪伴机器人** 子方向标识 | 温度、情感、共生 |
| 🌍 地球 | 地球 | **场地 / 现场** 标识 | 落地、现实、连接 |
| ♂ 火星 | 火星 | **具身智能 / 机器人** 子方向标识 | 探索、突破、未来生命 |
| ♃ 木星 | 木星 | **AI 硬件** 子方向标识 | 厚重、能量、硬核 |
| ♄ 土星 | 土星 | **资本评委 / 战略资助** 标识 | 长期主义、生态、循环 |
| ♅ 天王星 | 天王星 | **创意特别奖 / 黑马奖** 标识 | 反常识、突破、Outlier |
| ♆ 海王星 | 海王星 | **海外回流团队 / 远方** 标识 | 远端、未知 |

### 3.3 行星视觉规范

```
风格：
  - 极简线稿 + 探月橙描边
  - 不用真实纹理（不是 NASA 风），不用 3D 渲染
  - 类似手绘星图 / 中世纪天文图的克制感

画法：
  - 圆形主体，描边 1.5-2px 探月橙
  - 部分行星可加一道极简光环（土星、天王星）
  - 不填充纹理，最多用极淡的暖白点状肌理
  - 月球可加一两个陨石坑（致敬 Moonbot Logo）

尺寸用法：
  - Hero 区：单颗超大行星（月球，主导视觉）
  - 赛道区：每条赛道对应一颗中等行星
  - 子方向卡片：每个子方向左上角小行星图标
  - 区块过渡：行星可作为锚点装饰元素

布局：
  - 不堆砌——每页最多 1-3 颗行星
  - 不对称——行星不居中放，靠边或错位
```

### 3.4 行星与赛道的对应


## 6. 交互与动效

### 6.1 入场动效

| 元素 | 动画 | 时长 / 缓动 | 延迟 |
|---|---|---|---|
| 月球（首屏） | 从下方升起 + 缓慢公转 | 2.4s 升起 / 120s 公转 | 0.3s |
| Frontier 主标 | 字符依次淡入 | 1.5s | 1.0s |
| 中文主标题 + 副标 | 整体淡入 | 1.0s | 1.4s |
| Slogan | 行级依次淡入 | 0.8s × 2 | 1.8s / 2.2s |
| 数字大卡片 | 数字滚动至目标值 | 1.5s | 进入视口触发 |
| 卡片入场 | 上移 20px + 淡入 | 0.8s | 进入视口触发 |
| 行星图标 | 旋转 + 浮现 | 1.2s | 进入视口触发 |
| 三天流程时间轴 | 横向滑入 | 0.8s × 3 | 进入视口触发 |
| 评委矩阵卡片 | 矩阵式渐次出现 | 0.5s × N | 进入视口触发 |

### 6.2 行星动效特殊处理

```
- 月球（Hero 主视觉）：极慢自转（120s/360°），几乎察觉不到，但有呼吸感
- 其他行星：进入视口时缓慢淡入 + 微旋转（60s/360°）
- 行星之间不互动（不做"绕轨道转"的复杂动画，避免花哨）
- 鼠标 hover 行星：缩放 1.05 + 描边变粗 + 0.3s 过渡
```

### 6.3 自定义鼠标

- 圆点：10×10px / 探月橙 / 直接跟随
- 圆环：36×36px / 半透明探月橙 / 惯性追随（lerp 0.12）
- 悬停态：圆点 scale(2.5) + 圆环扩至 56px

### 6.4 滚动节奏

- scroll-snap 让每个 Section 滚动停顿
- 滚动指示器：首屏底部"向下滚动"中文提示

## 7. 必须遵守的禁止清单

- ❌ 蓝、绿、紫、红等其他色彩
- ❌ 圆角
- ❌ 渐变背景
- ❌ 粒子动画 / 3D 翻转 / 装饰元素堆砌
- ❌ "最大、最热、最潮"等娱乐化话术
- ❌ 表情符号 / Emoji（行星可用 unicode 符号但应替换为线稿 SVG）
- ❌ 除"Frontier"和"Physical AI"外的英文标题、英文标签
- ❌ 多个赞助方 Logo 大小并列（探月主标必须独大）
- ❌ 探月联盟具体成员名单 / 被投项目 / 估值数据 / 直投基金 等保密信息
- ❌ 任何会让品牌温暖友好化的元素（圆角、暖色、可爱图标等）
- ❌ **行星不用 NASA 真实纹理 / 不用 3D 渲染 / 不用 Disney 风格的拟人化处理**
