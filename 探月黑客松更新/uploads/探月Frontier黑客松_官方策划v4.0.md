# 第一部分 · 关于 Frontier

## 1. 什么是 Frontier

**Frontier** —— 边界。

不是已经画好的边界，是**还没画出的那条线**。

Physical AI 正处在这样一个时刻：技术路线刚刚收敛（VLA、世界模型、模仿学习正成为主流），但产业落地还在试错；学术正在出顶会论文，但能跑通真实世界的 demo 屈指可数；资本正在大规模涌入，但还没人说得清下一个 30 亿估值长在哪儿。

不只是机器人——AI 可穿戴、陪伴机器人、AI Pin、智能家居……所有让 AI 走出屏幕、进入物理世界的方向，都正站在同一条边界上。

这就是 Frontier 的位置——**在已知和未知的边界上**。

我们办这场黑客松，不是为了"做出一个产品"，是为了**让顶级的人在这条边界上集中工作 48 小时，看看能跑通什么**。

> **不为抵达，只为看清那条还没画出的线。**

## 2. 我们要做一场什么样的黑客松

### 2.1 不是什么

- ❌ 不是大众化的开发者狂欢
- ❌ 不是流量平台主导的内容工厂
- ❌ 不是政府背书的形象工程

## 3. 赛道设计：两条主赛道覆盖 Physical AI 全谱系

> 决策：**两条主赛道**，让评审与传播都聚焦。

### 主赛道 A · Physical AI（30 支队，占 60%）

> 让 AI 走出屏幕，进入物理世界。

**覆盖方向**：

| 子方向 | 范例 |
|---|---|
| **具身智能** | 机械臂操作 / 人形机器人 / 模仿学习 / VLA / 世界模型 |
| **AI 硬件 / 终端** | AI Pin / Rabbit R1 类终端 / AI 眼镜 / 智能音箱新形态 |
| **陪伴机器人** | Loona / Cozmo / Vector 类 / Replika 硬件化 / 桌面陪伴产品 |
| **AI 可穿戴** | 健康监测 / 情绪感知 / 脑机接口入门级 |
| **智能家居 / IoT** | 鸿蒙生态 / Matter 生态 / 多模态家居中枢 |

**赛前提供**：

- 主办方提供统一硬件套件池（机械臂 + 算力板 + 3D 相机 + 微型显示屏 + 麦克风阵列等），选手按项目方向自选
- 21 天线上 Bootcamp 免费技术培训（含硬件入门 + AI 模型部署）

**评审权重**（按技术难度分级）：

| 难度 | 技术路线 | 评审权重 |
|:-:|---|:-:|
| 1 级 | 目标检测 + 时间序列 / 简单交互硬件 | × 0.3 |
| 2 级 | 目标检测 + Agent / 多模态硬件 | × 0.5 |
| 3 级 | 模仿学习 / 端云协同 | × 0.8 |
| 4 级 | VLA / 世界模型驱动的硬件产品 | × 1.0 |

### 主赛道 B · AI Agent 与软件（20 支队，占 40%）

> 软件的极致，也是边界。

**覆盖方向**：

- 通用智能体（General Agent）
- 多模态系统（文 / 图 / 音 / 视）
- 世界模型应用（场景理解 / 规划）
- 行业 Agent（医疗 / 教育 / 法律 / 金融）
- AI 工具 / 创作 / 生产力软件

**特点**：纯软件赛道，门槛低，扩大全栈开发者参与池；鼓励"软件 × 硬件"跨界（用 LLM Agent 控制硬件设备）。

### Frontier 创意特别奖

不单独成赛道，从 A/B 中选拔。AI × 艺术 / AI × 生活 / AI × 交互创新。由弗兰克 + 媒体评审团评出，是破圈传播的核心素材。
